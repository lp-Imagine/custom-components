# 工具类组件库（自己工作中用到的一些和封装的组件）
## 相关技术栈 
* Vue3
* Vite
* Element Plus


# 安装
```bash
npm install @lp-imagine/custom-components
```

# 快速使用

## 组件说明
### components
```bash
import { components } from '@lp-imagine/custom-components'
```
### utils
```bash
import { utils } from '@lp-imagine/custom-components'
```
